# Introduction to Generative AI Hub in SAP AI Core
This hands-on course introduces developers to the world of Large Language Models (LLMs) in the Generative AI Hub of SAP AI Core.

You will also be introduced to SAP Cloud SDK for AI, the official Software Development Kit (SDK) for SAP AI Core. 

By using SAP Cloud SDK for AI, you can leverage the power of LLMs available in SAP's Generative AI Hub to create solutions that combine the power of LLMs with business context using SAP data.

<br>

- **Foundation models** are neural networks trained on large volumes of data using self-supervised learning that can be applied to many tasks.
- **Large language models (LLMS)** are a subcategory of foundation models for text incl. computer code.
- **Generative Al** can create novel output in text, images, sound, or video based on simple user input (called "prompts").
- **ChatGPT** is an application built on a large language model created by the vendor OpenAl. The model has been optimized for dialog.
- **Multi-modal models** can process prompts and generate output in various formats, including text, images, video, and speech.

### Introduction to the SAP Cloud SDK for AI

The SAP Cloud SDK for AI is the official toolkit for working with SAP AI Core, the Generative AI Hub, and the Orchestration Service. With this SDK, we can integrate chat completion into our applications, make use of prompt templating, apply grounding for accuracy, enable data masking for privacy, and add content moderation for compliance.

The SDK comes in three flavours: Python, Java and JavaScript.

To know more about the SAP Cloud SDK for AI, check out the [Official SAP Cloud SDK for AI documentation](https://sap.github.io/ai-sdk/docs/overview/overview-cloud-sdk-for-ai).
